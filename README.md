# What is this
A chart library base on vue and d3
## Usage （Unavailable due to being under development）
```
npm install vue-d3-chart 
```
## Develop your own chart library
```
git clone https://github.com/zzz945/vue-d3-chart.git
cd vue-d3-chart
npm install
npm run dev
```
